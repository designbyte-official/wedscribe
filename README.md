## Run Locally

**Prerequisites:**  Node.js (v18 or higher) and pnpm

1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key (optional, only needed for AI bio generation)
3. Run the app:
   ```bash
   pnpm run dev
   ```
